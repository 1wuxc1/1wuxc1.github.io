AGV 安全遥控系统设计文档（SX1278 收发机）

1. 系统架构

  • 发送端（遥控手柄）
    摇杆 / 开关 → MCU ADC/GPIO → 安全帧打包 → SPI 写入 SX1278 → 天线发射
    双路硬急停：双联急停按钮，一联常闭触点串联 SX1278 供电（物理断射频）；另一联常开触点接入 MCU 外部中断，触发后立即将 SPI 引脚（SCK/MOSI/NSS）设置为模拟输入或下拉，杜绝 IO 通过 ESD 二极管倒灌电流维持模块半工作状态。

  • 接收端（AGV 控制器）
    天线 → SX1278 → SPI 读出 → MCU 校验（同步字、软件 CRC、序列号）→ 字节序重组 → 摇杆归零校验 → 更新通道值 & 喂狗
    安全继电器驱动：MCU 输出 1kHz 连续 PWM 波，经 RC 电荷泵驱动动力继电器，任何 MCU 死机（输出恒高或恒低）均导致继电器自动断开。
    本地急停：串联于主接触器线圈回路，最高优先级。
    SBUS 输出：无论是否紧急状态，持续发送带 FAILSAFE 标志的 SBUS 帧，防止伺服驱动器因信号丢失而保持末态运行。

2. 硬件设计与接线

2.1 SX1278 模块引脚连接（模式0 SPI）

引脚 方向 连接 MCU 说明
VCC PWR 3.3V（经急停NC触点） 100µF 电解 + 100nF 陶瓷去耦
GND PWR GND 大面积敷铜
SCK IN SPI SCK
MOSI IN SPI MOSI 急停触发后必须下拉或设为模拟输入
MISO OUT SPI MISO
NSS IN 任意 GPIO 急停后同 MOSI 处理
RESET IN GPIO（推挽） 高电平有效，上电复位后保持高
DIO0 OUT GPIO（输入，可中断） 映射为 TxDone 或 RxDone

2.2 发送端双联急停电路

急停按下时：

  • SX1278 立即失电，射频消失。
  • MCU 中断服务函数将 SPI 相关引脚配置为模拟输入/下拉，防止漏电。

2.3 接收端安全继电器（PWM 电荷泵 可选）

原理：正常运行时，MCU 输出占空比 50% 的 1~5kHz PWM，电荷泵产生稳定的直流电平使三极管导通，继电器吸合。
若 MCU 死机（电平恒定），电容阻断直流，继电器自动断开，断开时间取决于 RC 时间常数，一般 < 20ms。

3.2 射频参数配置（工厂内推荐 433MHz）

关键指标计算：Payload 20 字节，BW=250kHz，SF=7，CR=4/5，前导 12 符号，硬件 CRC 开启时，空中包时间 (ToA) ≈ 27.8ms。

为防止高占空比导致模块过热，建议：

  • 发射周期 50ms 时，发射功率限 14dBm。
  • 若需 20dBm 满功率，发射周期应放宽至 100ms 以上。

注意事项：

  • 看门狗超时值建议根据发射周期调整：50ms 周期 → 300ms（6包）；100ms 周期 → 400~500ms。
  • 紧急状态下不仅切断继电器，还应通过 SBUS 强制通知伺服驱动器进入安全状态。

6.3 SBUS 编码与输出

SBUS 协议：100kbps，8E2，反相逻辑，帧长 25 字节。

  • 字节 0：起始符 0x0F
  • 字节 1~22：16 个通道，每个通道 11bit（范围 0~2047），对应 1000~2000μs 映射关系：
    sbus_val = (channel_us - 1000) * 2047 / 1000
  • 字节 23：标志位，bit3 为 FAILSAFE（0=正常，1=失效）
  • 字节 24：结束符 0x00

紧急状态下，所有通道强制设置为安全值（如运动通道 1024，即中位），并将 FAILSAFE 置位。

7. 完整安全机制汇总

安全机制 实现方式 响应时间
发送端急停 双联触点：物理断 SX1278 供电 + MCU 清理 SPI 引脚 ≤1ms
接收端通信丢失 软件看门狗，超时后停 PWM 继电器 + SBUS FAILSAFE ≤300ms（可配置）
MCU 死机/程序跑飞 PWM 电荷泵驱动继电器，恒电平即断开 <20ms
数据完整性 硬件 CRC + 软件 CRC16，双重校验 单包丢弃
重复帧攻击 序列号过滤，重包直接丢弃 实时
恢复安全 急停恢复后强制摇杆归零检测，否则不使能动力 首帧阻断
跨平台字节序陷阱 禁止 memcpy，统一用移位拼接 编译期安全
SBUS 链路丢失 紧急状态下持续发送 FAILSAFE 帧，通知伺服安全停机 ≤10ms（SBUS 周期）
本地物理急停 串联主接触器线圈，独立于软件 瞬时

8. 调试与验证建议

  1. 射频测试：先用 CW 模式验证发射功率和中心频率，确保 RegPaConfig 与预期 dBm 对应。
  2. ToA 验证：使用频谱仪抓取单次发射时长，确认约 28ms。
  3. 急停响应：示波器同时抓取发送端急停按钮触点和接收端继电器触点，确认断电时间 < 350ms。
  4. PWM 继电器：停止 PWM，测量继电器线圈电压跌落时间，确保 < 20ms。
  5. SBUS 一致性：逻辑分析仪解码 SBUS 帧，检查紧急状态下通道值归中且 FAILSAFE 置位。
