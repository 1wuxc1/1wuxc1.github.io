主极磁场、电枢磁场与气隙磁场

在电机（尤其是直流电机和同步电机）的分析中，主极磁场、电枢磁场和气隙磁场是三个核心概念。它们共同决定了电机的感应电动势、电磁转矩及运行性能。

1. 基本定义

磁场名称 产生来源 存在条件
主极磁场 由安装在主磁极上的励磁绕组通入直流电流（励磁电流）产生。 电机励磁后即存在，与负载无关
电枢磁场 由电枢绕组中流过的负载电流产生。 仅在电机带负载时出现
气隙磁场 主极磁场与电枢磁场在气隙中的合成磁场。 空载时等于主极磁场；负载时为合成磁场

2. 主极磁场（Main Pole Magnetic Field）

2.1 产生与磁路

主极磁场由 励磁磁动势 Ff=NfIfF_f = N_f I_fFf​=Nf​If​ 建立，其磁通路径为：

主极铁心 → 气隙 → 电枢齿/槽 → 电枢轭 → 相邻异极性主极下的气隙 → 相邻主极铁心 → 定子轭（磁轭）

空载时，只有主极磁场存在，此时气隙磁密波形沿电枢表面呈近似矩形分布（计及齿槽效应后为平顶波）。

2.2 特点

  • 相对于主极轴线对称分布。
  • 轴线与主极中心线重合（直轴方向）。
  • 大小主要取决于励磁电流和气隙长度。

3. 电枢磁场（Armature Magnetic Field）

3.1 产生与轴线

当电枢绕组流过电流时，产生 电枢磁动势。若电刷位于几何中性线上，电枢磁动势的轴线与主极轴线正交，即处于 交轴 方向。

3.2 磁路特点

  • 电枢磁通经电枢铁心、气隙、主极极靴、定子轭闭合。
  • 每个主极下的电枢磁动势呈三角形分布，气隙磁密为马鞍形（考虑极间区域磁阻大时）。
  • 电枢磁场单独作用时，使气隙磁场发生畸变，称为 电枢反应。

3.3 电枢反应的主要影响

  • 磁场畸变：气隙磁密零点偏离几何中性线，物理中性线偏移。
  • 去磁或增磁效应：发电机状态交轴电枢反应通常呈去磁趋势；电动机状态可能增磁（与电刷位置及饱和有关）。
  • 换向恶化：磁场畸变导致换向元件中的电抗电势变化，易产生火花。

4. 气隙磁场（Air-gap Magnetic Field）

4.1 合成原理

负载时，气隙磁场是 主极磁动势 与 电枢磁动势 的合成效果：

Fδ=Ff+FaF_\delta = F_f + F_a Fδ​=Ff​+Fa​

对应的气隙磁密 BδB_\deltaBδ​ 由合成磁动势和气隙磁导决定。由于铁磁材料的非线性，合成不能简单线性相加，需用磁路计算方法。

4.2 空载与负载的区别

  • 空载：Fa=0F_a = 0Fa​=0，气隙磁场等于主极磁场，波形对称。
  • 负载：气隙磁场相对于主极轴线发生偏斜，波形畸变，幅值可能因饱和而下降（去磁效应）。

4.3 物理意义

气隙磁场直接参与机电能量转换：

  • 感应电动势：E=CeΦδnE = C_e \Phi_\delta nE=Ce​Φδ​n
  • 电磁转矩：T=CTΦδIaT = C_T \Phi_\delta I_aT=CT​Φδ​Ia​

其中 Φδ\Phi_\deltaΦδ​ 为每极合成气隙磁通。

5. 三者关系总结

  1. 空载工况
    电枢磁场为零，气隙磁场 = 主极磁场。气隙磁密波形对称，物理中性线与几何中性线重合。

  2. 负载工况
    电枢磁场出现，气隙磁场 = 主极磁场 + 电枢磁场（矢量合成）。
    气隙磁场发生 电枢反应，物理中性线偏移，可能出现局部饱和。

  3. 补偿措施
    为抵消电枢反应的影响，大型电机常设置 补偿绕组（分布在主极极靴中）或 换向极，以改善换向并使气隙磁场恢复对称。

6. 数学描述（以直流电机为例）

  • 主极磁动势（每极）：Ff=NfIfF_f = N_f I_fFf​=Nf​If​
  • 交轴电枢磁动势（电刷在几何中线，单波/单叠绕组）：
    FaF_aFa​ 沿电枢表面线性变化，极尖处最大值为 NIa2a⋅τ2\frac{N I_a}{2a} \cdot \frac{\tau}{2}2aNIa​​⋅2τ​（具体依绕组形式而定）。
  • 合成气隙磁动势：
    Fδ(x)=Ff(x)+Fa(x)F_\delta(x) = F_f(x) + F_a(x)Fδ​(x)=Ff​(x)+Fa​(x)，其中 xxx 为沿电枢表面的位置。

在忽略饱和时，气隙磁密 Bδ(x)=μ0Fδ(x)δ′B_\delta(x) = \mu_0 \frac{F_\delta(x)}{\delta'}Bδ​(x)=μ0​δ′Fδ​(x)​，δ′\delta'δ′ 为等效气隙长度。计及饱和后需用磁化曲线迭代求解。

7. 工程意义

  • 理解和计算气隙磁场是电机设计的核心任务。
  • 电枢反应的程度直接影响 电压调整率（发电机）和 转矩-转速特性（电动机）。
  • 无补偿电机的气隙磁场畸变可能限制电机容量，从而催生补偿绕组和全补偿设计。

通过清晰区分这三个磁场，可以更系统地分析电机的稳态和瞬态行为，为电机控制和故障诊断提供理论依据。

功率角（Power Angle / Torque Angle）

功率角是同步电机（含电励磁与永磁）分析与控制中的核心概念，它直接反映了机电能量转换的强度与稳定性裕度。在经典同步电机理论中，功率角通常指转子励磁磁动势（或励磁电动势）与定子电枢反应合成磁场（或端电压）之间的相位差；在永磁同步电机矢量控制中，也常被称作转矩角。

1. 基本定义

功率角 δ\deltaδ 的两种等价表述：

  • 磁场空间角度：转子励磁磁场轴线与气隙合成磁场轴线之间的空间电角度。
  • 时间相位角度：励磁电动势 E˙0\dot{E}_0E˙0​ 与端电压 U˙\dot{U}U˙ 之间的相角差（忽略定子电阻时，也等于 E˙0\dot{E}_0E˙0​ 与合成电动势的相角）。

在稳态同步旋转坐标系中，δ\deltaδ 即为 q 轴与端电压矢量（或无穷大母线电压）之间的夹角，一般规定发电机状态下 δ>0\delta > 0δ>0，电动机状态下 δ<0\delta < 0δ<0（因功率流向不同）。

2. 功率角与电磁功率的关系（隐极同步电机）

对于隐极同步电机（Xd=Xq=XsX_d = X_q = X_sXd​=Xq​=Xs​），忽略电枢电阻，有功功率和电磁转矩的表达式为：

有功功率

P=E0UXssin⁡δP = \frac{E_0 U}{X_s} \sin\delta P=Xs​E0​U​sinδ

  • E0E_0E0​ —— 励磁电动势（每相有效值）
  • UUU —— 端电压（每相有效值）
  • XsX_sXs​ —— 同步电抗
  • δ\deltaδ —— 功率角

电磁转矩

T=3ωs⋅E0UXssin⁡δT = \frac{3}{\omega_s} \cdot \frac{E_0 U}{X_s} \sin\delta T=ωs​3​⋅Xs​E0​U​sinδ

其中 ωs\omega_sωs​ 为同步角速度。

无功功率

由上述公式可见，功率角 δ\deltaδ 直接决定了有功功率的大小和方向：

  • δ=0∘\delta = 0^\circδ=0∘：P=0P=0P=0，电机处于空载状态；
  • 0∘<δ<90∘0^\circ < \delta < 90^\circ0∘<δ<90∘：发电机状态，P>0P>0P>0（发出有功）；
  • −90∘<δ<0∘-90^\circ < \delta < 0^\circ−90∘<δ<0∘：电动机状态，P<0P<0P<0（吸收有功）；
  • δ=±90∘\delta = \pm 90^\circδ=±90∘：理论最大功率极限（静态稳定极限）。

3. 凸极同步电机的功率角特性

凸极电机（Xd≠XqX_d \neq X_qXd​=Xq​）的功率表达式包含一个附加磁阻功率分量：

P=E0UXdsin⁡δ+U22(1Xq−1Xd)sin⁡2δP = \frac{E_0 U}{X_d} \sin\delta + \frac{U^2}{2}\left(\frac{1}{X_q} - \frac{1}{X_d}\right) \sin 2\delta P=Xd​E0​U​sinδ+2U2​(Xq​1​−Xd​1​)sin2δ

  • 第一项：励磁功率，与 sin⁡δ\sin\deltasinδ 成正比；
  • 第二项：磁阻功率，与 sin⁡2δ\sin 2\deltasin2δ 成正比，由交直轴磁路不对称产生。

对应的转矩特性 T=P/ωsT = P/\omega_sT=P/ωs​ 不再是以 90∘90^\circ90∘ 为峰值的纯正弦曲线，最大转矩点前移（δ<90∘\delta < 90^\circδ<90∘）。

4.1 时空相矢图

  • 以端电压 U˙\dot{U}U˙ 为参考相量；
  • 励磁电动势 E˙0\dot{E}_0E˙0​ 超前 U˙\dot{U}U˙ 的角度即为功率角 δ\deltaδ；
  • 电枢反应磁动势 F⃗a\vec{F}_aFa​ 与转子励磁磁动势 F⃗f\vec{F}_fFf​ 共同作用，使气隙合成磁动势 F⃗δ\vec{F}_\deltaFδ​ 与 U˙\dot{U}U˙ 相对应；

4.2 物理类比

功率角类似于弹簧-质量系统中弹簧的拉伸角度：

  • 机械输入转矩（原动机）使转子磁场“前拉”气隙磁场，角度增大，输出有功增加；
  • 若超过弹性极限（δ>90∘\delta > 90^\circδ>90∘ 或凸极机更小角度），同步能力丧失，电机失步。

5. 永磁同步电机中的功率角（转矩角）

在永磁同步电机（PMSM）的矢量控制（id=0i_d = 0id​=0 控制或最大转矩电流比控制）中，功率角的概念常延展为 转矩角 θT\theta_TθT​：

  • 定义：转子永磁体磁场（d轴）与定子电流矢量 i⃗s\vec{i}_sis​ 之间的夹角。
  • 在 d−qd-qd−q 坐标系中，若定子电流矢量 isi_sis​ 与 q 轴夹角为 β\betaβ（电流相位角），则电磁转矩：

    T=32p[ψfiq+(Ld−Lq)idiq]T = \frac{3}{2} p [ \psi_f i_q + (L_d - L_q) i_d i_q ] T=23​p[ψf​iq​+(Ld​−Lq​)id​iq​]

    其中 id=−issin⁡βi_d = -i_s \sin\betaid​=−is​sinβ，iq=iscos⁡βi_q = i_s \cos\betaiq​=is​cosβ，ppp 为极对数，ψf\psi_fψf​ 为永磁磁链。

当 id=0i_d = 0id​=0 时，转矩与 iqi_qiq​ 成正比，β=0\beta = 0β=0，此时转子磁场与定子电流产生的电枢反应磁场在空间正交，类似于传统同步电机的 δ=90∘\delta = 90^\circδ=90∘ 工况；若采用弱磁控制，β\betaβ 增大，转矩角变化。

在 PMSM 的功角特性中，若将 δ\deltaδ 重新定义为 ψf\psi_fψf​ 与定子合成磁链的夹角，则传统功率角公式同样适用，但需注意磁路饱和与凸极效应。

6. 静态稳定与动态稳定

6.1 静态稳定条件

同步电机稳态运行的必要条件：

dPdδ>0\frac{dP}{d\delta} > 0 dδdP​>0

即功率-角度特性的斜率为正。对隐极机，δ<90∘\delta < 90^\circδ<90∘；凸极机则小于略小于 90∘90^\circ90∘ 的某一角度。

过载能力（静态过载倍数）：

km=Pmax⁡PN=1sin⁡δNk_m = \frac{P_{\max}}{P_N} = \frac{1}{\sin\delta_N} km​=PN​Pmax​​=sinδN​1​

其中 δN\delta_NδN​ 为额定功率角（通常隐极发电机取 30∘∼40∘30^\circ\sim40^\circ30∘∼40∘，稳态裕度充足）。

6.2 动态过程

当系统受到大扰动（短路、切机等）时，功率角会发生振荡，是否失步取决于故障清除时间和转动惯量等。等面积定则是分析暂态稳定的基本方法。

7. 功率角的测量与应用

  • 实际测量：通过转子位置传感器（旋变、光电编码器）检测转子磁极位置，与定子电压相位比较得到 δ\deltaδ。
  • 控制应用：
    • 电网并联运行时，调节原动机转矩改变 δ\deltaδ，控制有功输出；
    • 励磁电流调节无功功率，但对 δ\deltaδ 有间接影响（E0E_0E0​ 改变影响 PPP-δ\deltaδ 曲线幅值）；
    • 在 PMSM 高性能驱动中，实时估算或测量功率角/转矩角以实现弱磁、MTPA 控制。

8. 总结

  • 功率角 δ\deltaδ 是同步电机耦合转子机械系统与定子电磁系统的关键状态量，其大小直接决定电磁转矩和有功功率。
  • 隐极机：P∝sin⁡δP \propto \sin\deltaP∝sinδ，静态稳定极限 δ=90∘\delta = 90^\circδ=90∘。
  • 凸极机：附加磁阻转矩使特性复杂化，极限角前移。
  • 永磁电机：功率角/转矩角概念扩展至电流矢量角，对矢量控制和弱磁运行至关重要。

深入理解功率角的物理本质与数学模型，是进行电机设计、系统稳定性分析和高性能控制的基础。
